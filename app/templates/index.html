ALL-CP-PLATFORMS-API

Documentation

An API to get user Information from various online programming platforms.

  • Request Format:


    http://all-cp-platforms-api.herokuapp.com/api/{platform}/{username}

    Replace {platform} with the name of platform and {username} with the username of the user



  • Supported Platforms:

    • CodeChef
    • Codeforces
    • Atcoder
    • Spoj

Note: this project in under progress. I will add more platforms soon.
